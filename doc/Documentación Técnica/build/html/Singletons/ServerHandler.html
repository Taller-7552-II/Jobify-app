ServerHandler¶

public class ServerHandler¶

Permite realizar operaciones HTTP a través de una API Rest, utilizando un token de authenticación. Es importante mencionar, que estas operaciones NO deben realizarse en la UI thread de Android, sino que deben ser desplazadas a otra thread. En esta aplicación, se utilizan AsyncTasks para ejecutar las operaciones del ServerHandler en la Background thread.
